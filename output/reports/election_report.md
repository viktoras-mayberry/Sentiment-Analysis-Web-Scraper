## Methodology

### Data Collection

Data was collected from multiple platforms using platform-specific methods:

| Platform | Method | Authentication |
|----------|--------|----------------|
| Nairaland | HTML parsing (BeautifulSoup) | None required |
| Reddit | PRAW (official API wrapper) | OAuth (client credentials) |
| Twitter/X | Tweepy (API v2) | Bearer token |
| Facebook | Graph API (requests) | Access token |

### Candidate Identification

Candidates were identified dynamically using a two-layer approach:

1. **Regex-based name extraction** - Sequences of 2-3 capitalized words matching name patterns (handles hyphenated Nigerian names)
2. **Frequency + context filtering** - Names must appear 2+ times near election-related keywords, with cross-platform validation bonus

### Sentiment Analysis

Primary model: **transformers**

- Three-tier fallback: HuggingFace Transformers > VADER > TextBlob
- Scores range from -1.0 (most negative) to +1.0 (most positive)
- Handles English and Nigerian Pidgin text

### Assumptions and Limitations

- Only public data is collected; no private accounts or groups are accessed
- Author identities are anonymized using SHA-256 hashing
- Sentiment models may misclassify sarcasm or culturally-specific expressions
- Twitter API free tier does not support search; results depend on API access level
- Facebook Graph API has restricted public post search since 2018
- Candidate identification may miss names not following standard capitalization

### Ethical Compliance

- All data sources are public and freely accessible
- robots.txt compliance is respected where applicable
- Request delays are enforced to avoid server overload
- No automated logins or credential stuffing is performed
- Author data is anonymized before storage
